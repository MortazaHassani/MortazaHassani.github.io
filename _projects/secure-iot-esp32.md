This project implements secure IoT communication on ESP32 using lightweight cryptography and MQTT.

**Features:**
- Encrypted data transfer over insecure channels
- Lightweight cryptography workflow for constrained devices
- Real-time data visualization web panel
- Docker and Telegram API integration for automated notifications

**Technologies:** ESP32, MQTT, Docker, Python, Telegram API, lightweight cryptography
