This project designs and implements a **secure end-to-end communication protocol** for IoT devices based on the ESP32 microcontroller, using the **SPECK lightweight block cipher** — specifically designed for constrained devices.

**Features:**
- SPECK cipher implementation optimized for ESP32
- Secure key exchange and session management
- Real-time monitoring dashboard via Telegram bot integration
- Docker-based logging and alerting infrastructure
- Energy consumption analysis and benchmarking

**Technologies:** ESP32, C/C++, SPECK cipher, Docker, Python, Telegram API
