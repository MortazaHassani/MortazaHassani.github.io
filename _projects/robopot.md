**Best Senior Design Project Award** recipient at Mohammad Ali Jinnah University.

RoboPot is a fully autonomous robot that uses **computer vision** and **PID control** to locate, navigate to, and pot golf balls on a miniature course. Built entirely by the team as the capstone project.

**Features:**
- Real-time ball detection using OpenCV on Raspberry Pi
- PID-based motor control for precise positioning
- MQTT-based wireless telemetry and monitoring
- Custom mechanical arm for ball engagement
- Fully autonomous operation from detection to potting

**Technologies:** Raspberry Pi, Python, OpenCV, MQTT, PID control, stepper motors
